# 공 배틀 V61

친구들과 링크로 플레이할 수 있는 공 배틀 웹게임입니다.

## 실행

GitHub Pages 주소에 접속하면 `index.html`이 자동으로 실행됩니다.

## 업데이트 방법

1. 새 공 배틀 HTML 파일을 다운로드합니다.
2. 파일 이름을 `index.html`로 변경합니다.
3. 이 저장소의 기존 `index.html`을 새 파일로 교체합니다.
4. 변경 사항을 Commit합니다.
5. 잠시 기다린 뒤 기존 GitHub Pages 주소를 새로고침합니다.

게임 주소는 업데이트해도 바뀌지 않습니다.
